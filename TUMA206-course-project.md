# TUMA206 Course Project

## Project Title

Water Treatment Plant Digital Twin with Fault-Aware AI Operator Assistant

## Course Context

This repository is the course project for TUMA206 Modern Developments in Industry.
The project develops a working digital twin of a water treatment plant, including a
process simulator, PLC-style control logic, MQTT messaging, historian storage, an
operator dashboard, fault injection, fault detection, and an AI-supported operator
assistant.
